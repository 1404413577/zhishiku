## 15. 建议自动化测试范围

### 单元测试

- 文档 normalize。
- 文档树构建。
- 文档移动合法性。
- Markdown 双链提取。
- Markdown 摘要生成。
- 搜索索引增删改。
- AI 错误归一化。
- 设置合并逻辑。

### E2E 测试

- 新建文档到保存。
- 编辑文档到搜索。
- 新建文件夹并移动文档。
- 搜索并打开文档。
- AI 对话归档为文档。
- 导出和导入数据。
