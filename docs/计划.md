一、编辑体验增强
1. 文档目录大纲 (TOC 侧边栏)
已有 tocDebug.js 工具，提取 Markdown 标题生成浮动导航，长文档点击跳转。阅读模式和编辑模式都可用。

2. 专注模式
一键隐藏工具栏、侧边栏，只保留编辑器，背景变暗/居中窄栏，适合长时间写作。已有暗色模式可复用。

3. 编辑/预览分屏
双栏并排显示，左侧编辑右侧实时预览，之前代码里留了 split-view 的 CSS 痕迹但未实现。

4. Markdown 表格可视化插入
点击格子选择行列数插入表格，比手写 |--|--| 方便很多。纯 UI 交互。

二、文档操作
5. 单文档导出 PDF / HTML / MD 文件
浏览器 window.print() 调起打印为 PDF，或 Blob 下载为 .md/.html 文件。纯客户端，无需任何后端。

6. 批量操作
列表多选后批量打标签、批量删除、批量导出。目前只能单文档操作。

7. 文档模板
预设几个模板（会议纪要、周报、读书笔记、每日日志），新建时选择模板快速开始。本质就是预设 Markdown 内容。

三、数据可视化与发现
8. 词云图
从所有文档内容提取高频词，生成词云展示知识库主题分布。需引入 echarts-wordcloud 扩展。

9. 随机发现
首页加个"随便看看"按钮，随机导航到一篇文档，适合回顾旧笔记。

10. 阅读时间估算
根据文档字数自动显示"预计阅读 X 分钟"，在文档卡片和阅读页显示。

四、效率工具
11. 全局快捷键面板
按 ? 弹出快捷键一览，统一管理已有的 Ctrl+S 保存等快捷键。

12. 快捷笔记入口
导航栏常驻一个"快速记录"按钮，弹出小型编辑框，不打乱当前工作流，适合捕获临时想法。
